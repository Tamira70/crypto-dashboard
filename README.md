# 📊 Crypto Dashboard

Ein modernes Crypto Dashboard mit Live-Daten aus der CoinGecko API.  
Entwickelt mit TypeScript und Vite und deployed auf einem eigenen Linux-Server.

---

## 🚀 Features

- 🔄 Live API Integration (CoinGecko)
- 🔍 Suche nach Coins & Symbolen
- 📈 24h Preisentwicklung
- 💰 Marktstatistiken (Market Cap)
- ⚡ Schnelles Frontend mit Vite
- 🌐 Deployment auf eigenem Server (Apache)

---

## 🛠️ Tech Stack

- TypeScript
- Vite
- HTML / CSS
- REST API (CoinGecko)
- Linux Server (Deployment)
